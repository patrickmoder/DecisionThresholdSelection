In an effort to classify an object, an Artificial Intelligence (AI) algorithm gets trained to make such classifications. The AI algorithm has a binary output that indicates whether an object belongs to the class of interest (positive) or not (negative). Some examples are classifying an object as A or B, black or white, big or small, and so on.

When the AI algorithm makes a correct classification, we therefore differentiate between two outcomes:

  • True Positive (TP): the AI algorithm correctly predicts a positive class.
  • True Negative (TN): the AI algorithm correctly predicts a negative class.

Although being properly trained on this task, the AI algorithm is not entirely perfect and makes false classifications occasionally. Such false predictions, or misclassifications, can appear in two different flavors:

  • False Positive (FP): the AI algorithm predicts a positive class but the object belongs to the negative class.
  • False Negative (FN): the AI algorithm predicts a negative class but the object belongs to the positive class.
